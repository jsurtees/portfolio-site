Font Design
Quantum

About

Quantum shows off a futuristic style. It is modelled off round geometric fonts like Futura and Avante Garde Gothic. These features make the font clean and readable. They work well with the fonts mono-weight feature. This font reuses a lot of shapes as it is mostly made of straight lines and circles. This makes it visually appealing and cohesive. All of the letters shown are partly built from identical shapes. Quantum is unmistakably a set.

Sketching

The sketching stage for this project was a lot of fun. I enjoyed drawing words and letters in different styles with new markers and brushes. However I decided to continue with a geometric font and learn from popular sans serifs like Futura and Avant Garde.

An Alphabet

Sketches were brought into illustrator and each letter vectorized. Each letter was built using guides for consistent measurements. At this stage some mock text was put together for peer review. Minor changes were made for better readability.

Sample Text

This is a pangram to display the font in action. The next stage was to bring the illustrator files into glyphs, clean up extra anchor points, kern and export.

Posters

These are the graphics used for the type posters. They show the fonts anatomy, mono-weight and geometric/repeating shapes.
